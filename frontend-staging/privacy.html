Privacy Policy

Last updated: November 9, 2025

1. Information We Collect

Demo Usage:

  • Session ID (cookie-based, anonymous)
  • IP address (for rate limiting)
  • User agent (browser information)
  • Chat messages (stored locally, not shared)

Waitlist:

  • Email address (only if you join waitlist)
  • Signup timestamp
  • IP address and referrer (optional, for analytics)

2. How We Use Your Information

  • Rate Limiting: Prevent abuse (20 queries/day per session)
  • Service Delivery: Store chat history for session persistence
  • Waitlist Communication: Send beta launch notifications (opt-in only)

3. Data Storage

  • Location: DigitalOcean VPS (Frankfurt, Germany)
  • Database: SQLite (local file storage)
  • Retention: Demo sessions expire after 30 days
  • Backups: Daily automated backups (7-day retention)

4. Third-Party Services

We use AI providers to process your questions:

  • Google (Gemini): Free tier uses Gemini API
  • OpenAI (GPT-4): PRO tier only (coming Q1 2026)
  • Anthropic (Claude): PRO tier only (coming Q1 2026)

Important: Your messages are sent to these AI providers for processing. We do NOT store AI provider API keys. Read their privacy policies:

  • Google Privacy Policy
  • OpenAI Privacy Policy
  • Anthropic Privacy Policy

5. Cookies

We use ONE cookie:

  • chika_demo_session: Anonymous session tracking (30-day expiry)
  • Purpose: Rate limiting and chat history
  • Type: HttpOnly, SameSite=Lax (secure)

6. Your Rights (GDPR)

If you're in the EU, you have rights to:

  • Access: Request your data (email us)
  • Delete: Request deletion (we'll remove within 30 days)
  • Export: Get a copy of your data
  • Opt-out: Unsubscribe from emails anytime

7. Data Security

  • HTTPS encryption (TLS 1.3)
  • HttpOnly cookies (XSS protection)
  • Input sanitization (SQL injection prevention)
  • Rate limiting (abuse prevention)

8. Children's Privacy

CHIKA is NOT intended for users under 13. We do not knowingly collect data from children.

9. Changes to This Policy

We'll update this page if our practices change. Last updated date shown at top.

10. Contact

Questions? Email: privacy@chika.page

Or open an issue: GitHub Issues
